# Buto-Plugin-ElementFullscreen
Transform a div element to full screen from an icon button.
## Embed CSS and Javascript.
```
type: widget
data:
  plugin: element/fullscreen
  method: embed
```
## Button
```
type: widget
data:
  plugin: 'element/fullscreen'
  method: button
  data:
    id: _id_of_div_to_toogle_fullscreen_
```
